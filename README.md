# Eastvantage – Data Engineer Assignment

## Scenario

Company XYZ ran a promo sale for signature items **x, y, z**. The goal is to extract total quantities purchased per customer aged **18–35** and support a targeted marketing strategy.

---

## Schema

```
Customer          Sales             Orders            Items
──────────        ──────────        ──────────────    ──────────
customer_id (PK)  sales_id (PK)     order_id (PK)     item_id (PK)
age               customer_id (FK)  sales_id (FK)     item_name
                                    item_id (FK)
                                    quantity (nullable)
```

**Business rules:**
- A sales receipt can have multiple items.
- Clerks record **all** items per order; items not bought are stored as `quantity = NULL`.
- A customer can have multiple sales transactions.

---

## Solution

Two approaches are provided in `solution.py`:

| Approach | Description |
|----------|-------------|
| **SQL** | Single aggregation query — `JOIN` + `GROUP BY` + `HAVING` |
| **Pandas** | Load raw tables → merge → `groupby` + `sum` |

Both produce identical output: a semicolon-delimited CSV.

## SQL Approach – Query Explained

```sql
SELECT
    c.customer_id   AS Customer,
    c.age           AS Age,
    i.item_name     AS Item,
    CAST(SUM(o.quantity) AS INTEGER) AS Quantity
FROM   Customer c
JOIN   Sales    s ON s.customer_id = c.customer_id
JOIN   Orders   o ON o.sales_id    = s.sales_id
JOIN   Items    i ON i.item_id     = o.item_id
WHERE  c.age BETWEEN 18 AND 35
  AND  o.quantity IS NOT NULL       -- exclude items not bought
GROUP  BY c.customer_id, c.age, i.item_id, i.item_name
HAVING SUM(o.quantity) > 0          -- omit items with zero total
ORDER  BY c.customer_id, i.item_name;
```

Key decisions:
- `WHERE quantity IS NOT NULL` filters rows where the clerk recorded a non-purchase before aggregation.
- `HAVING SUM(...) > 0` removes any item whose quantities sum to zero after aggregation.
- `CAST(... AS INTEGER)` ensures no decimal points in the output.

---

## Pandas Approach – Steps

1. Load all four tables into DataFrames.
2. Filter `Customer` to `age BETWEEN 18 AND 35`.
3. Merge `Customer → Sales → Orders → Items`.
4. Drop rows where `quantity IS NULL` (`dropna`).
5. `groupby(['customer_id', 'age', 'item_name']).sum()`.
6. Filter out rows where the summed quantity equals zero.
7. Cast `quantity` to `int`.
8. Rename columns and sort.
